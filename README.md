## Algorithms

The project is implemented with Vite.
To run the Vit program, install vite and use the following commands:

**npm install**

**npm run dev**

Implementation of search issues
- binary search
- hash table
- find min max
- comparative sorting
    - bubble sort
    - insertion sort
    - selection sort
    - merge sort
    - quick sort
    - heap sort
    - tree sort with bst
    - tree sort with AVL
- non comparative sorting
    - counting sort
    - radix sort
    - bucket sort
      
Implementation of dynamic programming

- fibonancci
- multiplication of matrices
- floyd
- back pack 0/1
